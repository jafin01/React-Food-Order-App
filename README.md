## Technology

The application is built with:

- React.js
  - useState
  - useEffect
  - ReactPortal
  - useRef
  - forwardRef
  - context API
  - useReducer
  - CSS modules
  - react fragments and
  - reusable components

## Features

- Add food item to the cart in frontend 
- Update quantity
- Remove cart items
